hippie
======

a hypermedia api framework

Features: 

  * Generic domain style
    * Item
    * Collection
    * Paged collection
  * Linkes factor: dereferenceable uri
  * Context(schema) to UI render template
  * Custom meta data
  * OAuth2 (Password, Authorization code flow), support client specifed scope
